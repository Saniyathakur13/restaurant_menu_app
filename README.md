## ✨ Features

### Customer Features
- 🍽️ **Browse Menu** - View all available dishes with descriptions
- 🔍 **Search & Filter** - Search by name, filter by Veg/Non-Veg
- 🛒 **Shopping Cart** - Add/remove items, update quantities
- 💰 **Price Calculation** - Automatic tax (18%) and delivery charges
- 📦 **Order Checkout** - Collect delivery details and payment
- 💳 **Payment Integration** - Cash on Delivery & Razorpay support
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile

### Admin Features (Future)
- 📊 **Order Management** - View and manage customer orders
- 🍕 **Menu Management** - Add/Edit/Delete menu items
- 📈 **Sales Analytics** - Track orders and revenue

## 🛠️ Tech Stack

### Backend
- **Java 17** - Programming language
- **Spring Boot 3.x** - Framework
- **Spring MVC** - Web layer
- **Spring Data JPA** - Database operations
- **H2/MySQL** - Database

### Frontend
- **HTML5** - Structure
- **CSS3** - Styling with animations
- **JavaScript (ES6)** - Interactivity
- **Thymeleaf** - Template engine (optional)

### Libraries & Tools
- **Font Awesome 6** - Icons
- **Google Fonts (Poppins)** - Typography
- **Razorpay** - Payment gateway
- **Maven** - Dependency management
- **Git** - Version control
